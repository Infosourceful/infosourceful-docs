# Private Documentation Area

This directory contains internal reference documents that are **not published** to the public documentation site.

## Purpose

Files in this directory are:
- Available in the repository for team reference
- Excluded from MkDocs builds
- Not accessible on the public documentation site

## Contents

- `color-palette-reference.md` - Application color palette reference for matching MkDocs theme

## Adding Files

To add new internal-only documents:
1. Place them in this `_private/` directory
2. Files starting with `_` are automatically excluded by MkDocs
3. They will not appear in the navigation or be built into the site
